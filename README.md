# Exampion
*Conquer your examination of conscience.*

Exampion is a trusty assistant for your nightly examen in the form of a Discord bot.

Every night at 10pm, Exampion starts its review, where it asks you to score each habit or virtue, giving a final score at the end. If you don't repond within an hour, then Exampion notifies your accountability partner, so you make sure to do it!

## Development

Clone this repository and develop inside a VS Code devcontainer. All dependencies are installed with `uv` upon building.

To launch and debug the program, press F5 or hit the green play button in the VS Code **Run and Debug** panel.

## Execution

The review runs on a nightly CronJob via GitHub Actions.
